Skyblock Display Name Mapper

Click the button below to attempt to fetch **all** item data from the NotEnoughUpdates GitHub repository, extract their display names and internal names, and combine them into a single JSON mapping for download.

**WARNING:** This script fetches data directly from GitHub's Git Tree API and raw content. It is subject to **rate limits** (e.g., 60 requests/hour) and potential **CORS issues**. If it fails, check your browser's console (F12) for network errors. Fetching all items can take a long time.

Click "Generate Display Name Map (All Items)" to begin.
